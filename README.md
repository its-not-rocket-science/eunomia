# Eunomia

**Eunomia** — Knowledge-Centric Reasoning AI  
Inspired by the Greek goddess of order and harmony, Eunomia combines structured ontological reasoning with commonsense flexibility.

## Features
- Ontology-driven logical reasoning
- Commonsense knowledge integration
- Hybrid architecture for adaptable AI

## Getting Started

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
